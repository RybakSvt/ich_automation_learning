# Homework 3: UI Automation Test for ITCareerHub

## Описание задания
Автоматизация тестирования пользовательского интерфейса сайта ITCareerHub с использованием Selenium WebDriver и Pytest.
Тест проверяет наличие ключевых элементов на странице и функциональность телефонной иконки.

## Файлы в папке
- `test_itcareerhub_ui.py` — основные тесты элементов главной страницы
- `test_itcareerhub_contact_modal.py` — тест функциональности телефонной иконки
- `README.md` — это описание
- `requirements.txt` — зависимости проекта

```bash
# Запуск всех тестов

# Запуск с подробным выводом
pytest -v
