# Homework 5: UI Automation Tests with Selenium - Iframes and Drag&Drop

## Описание задания
Автоматизация тестирования пользовательского интерфейса с работой во фреймах (iframes) и функциональностью перетаскивания элементов (Drag&Drop) с использованием Selenium WebDriver и Pytest.

## Файлы в папке hw5
test_iframe_content.py — тест проверки текста внутри iframe

test_image_to_trash.py — тест перетаскивания изображения в корзину

README.md — это описание

requirements.txt — зависимости проекта

## Как запустить тесты

# Запуск всех тестов
pytest

# Запуск с подробным выводом
pytest -v

# Запуск конкретного файла
pytest test_iframe_content.py -v
pytest test_image_to_trash.py -v

## Цель задания
Написать автоматизированные тесты для проверки работы с iframe и функциональности Drag&Drop на различных веб-страницах.

# Функциональность тестов

## test_iframe_content.py:

Автоматическое открытие браузера Chrome

Переход на страницу https://bonigarcia.dev/selenium-webdriver-java/iframes.html

Переключение в iframe на странице

Поиск и проверка наличия текста "semper posuere integer et senectus justo curabitur."

Проверка отображения текста на странице

Возврат к основному контенту страницы

Автоматическое закрытие браузера после теста

## test_image_to_trash.py:

Автоматическое открытие браузера Chrome

Переход на страницу https://www.globalsqa.com/demo-site/draganddrop/

Обработка всплывающего окна с согласием на куки

Переключение в iframe с Drag&Drop функциональностью

Захват первой фотографии из галереи

Перетаскивание фотографии в область корзины (Trash)

Проверка результатов:

В корзине должна быть одна фотография

В основной области должно остаться три фотографии

Возврат к основному контенту страницы

Автоматическое закрытие браузера после теста

Особенности реализации
Использование WebDriverWait для явных ожиданий элементов

Работа с ActionChains для выполнения Drag&Drop операций

Обработка iframe и переключение между контекстами

Блокировка всплывающих элементов и рекламы

Обработка динамических элементов страницы
