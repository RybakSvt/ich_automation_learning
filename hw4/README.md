# Homework 4: UI Automation Tests with Selenium

## Описание задания
Автоматизация тестирования пользовательского интерфейса двух различных веб-страниц с использованием Selenium WebDriver и Pytest. Тесты проверяют функциональность изменения текста кнопки и загрузки изображений.

## Файлы в папке hw4
- `test_text_input.py` — тест изменения текста кнопки
- `test_loading_images.py` — тест загрузки изображений и проверки атрибутов
- `README.md` — это описание
- `requirements.txt` — зависимости проекта

## Как запустить тесты
```bash
# Запуск всех тестов
pytest

# Запуск с подробным выводом
pytest -v

# Запуск конкретного файла
pytest test_text_input.py -v
pytest test_loading_images.py -v

Цель задания
Написать автоматизированные тесты для проверки функциональности веб-элементов на двух различных сайтах.

Функциональность тестов

test_text_input.py:
Автоматическое открытие браузера Chrome

Переход на страницу http://uitestingplayground.com/textinput

Ввод текста "ITCH" в поле ввода

Нажатие на синюю кнопку

Проверка изменения текста кнопки на "ITCH"

Автоматическое закрытие браузера после теста

test_loading_images.py:
Автоматическое открытие браузера Chrome

Переход на страницу https://bonigarcia.dev/selenium-webdriver-java/loading-images.html

Ожидание загрузки всех изображений

Получение значения атрибута alt у третьего изображения

Проверка, что значение атрибута alt равно "award"

Автоматическое закрытие браузера после теста

Автоматическая установка ChromeDriver через webdriver-manager
